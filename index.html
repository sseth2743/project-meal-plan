Type: pwd (shows current location) 3. Type: ls (shows files) 4. If you don't see index.html: - You're in wrong folder - Try: cd Downloads - Then: cd Project\ Meal\ Plan ``` --- ### **Problem: "git push" says "permission denied"** **Fix:** ``` 1. You entered wrong GitHub password 2. It's not the same as your computer password 3. Go to GitHub.com 4. Click your profile (top right) 5. Click "Settings" → "Developer settings" → "Personal access tokens" 6. Generate a new token 7. Use that token as password instead ``` --- ### **Problem: Page shows "404" when opening app link** **Fix:** ``` 1. Wait another 2-3 minutes 2. GitHub might still be publishing 3. Try refreshing (Ctrl+R) 4. Try clearing cache (Ctrl+Shift+Delete) 5. Try different browser ``` --- ### **Problem: Dashboard doesn't update when selecting meals** **Fix:** ``` 1. Hard refresh: Ctrl+Shift+R (not just Ctrl+R) 2. Clear browser cache: a. Press Ctrl+Shift+Delete b. Click "Clear data" 3. Close browser completely 4. Reopen browser 5. Go back to app ``` --- ### **Problem: Reorder links still show 404** **Fix:** ``` 1. The fix we applied: - Noon: https://www.noon.com/uae-en/grocery ✅ - Carrefour: https://www.carrefour.ae ✅ - Careem: https://www.careem.com/en-ae ✅ 2. If still 404: a. Check if your internet works b. Try opening links directly in browser c. These sites might be down (not our issue) d. Wait and try again later ``` --- ## 💡 TIPS ### **Tip 1: How to Get GitHub Password** ``` Don't use your computer password! Use a Personal Access Token: 1. Go to github.com 2.
